Integrate Atoms.Ninja into your workflow

Base URL

https://atoms.ninja/api/v1

Endpoints

POST /execute

Execute security tool command via Kali MCP

POST /api/execute
Content-Type: application/json

{
  "tool": "nmap",
  "args": ["-sV", "target.com"]
}

Response:
{
  "output": "..scan results..",
  "exitCode": 0
}

POST /ai/chat

Natural language command execution

POST /api/ai/chat
Content-Type: application/json

{
  "message": "scan example.com for vulnerabilities"
}

Response:
{
  "action": "execute",
  "command": "nmap -sV example.com",
  "explanation": "Port scanning..."
}

GET /tools

List available security tools

GET /api/tools

Response:
{
  "tools": [
    "nmap", "nikto", "sqlmap", "metasploit", ...
  ],
  "count": 500
}

Authentication

All requests require API key in header:

Authorization: Bearer YOUR_API_KEY
