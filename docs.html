Quick start guide for Atoms.Ninja

Step 1: Launch Terminal

Click "INIT TERMINAL" on the homepage to access the Atom AI interface. The terminal will initialize with Kali MCP server connection.

Step 2: Natural Language Commands

Talk to Atom like you would to Jarvis. Use natural language: 

"scan example.com"
"find vulnerabilities in 192.168.1.1"
"pwn target.com"
"what ports are open?"

Step 3: Direct Tool Commands

Or use direct Kali Linux commands: 

nmap -sV target.com
nikto -h http://target.com
sqlmap -u http://target.com?id=1
dirb http://target.com

Step 4: Genius Mode

Trigger autonomous attack chains:

  • exploit target.com - Full web app pentest
  • find entry point on target.com - Intelligent recon
  • use all methods on target.com - Complete arsenal
  • bypass waf on target.com - WAF evasion techniques
